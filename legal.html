Privacy

This site is designed to collect as little data as possible:

  • No cookies, no analytics, no trackers. No Google Fonts, no CDNs — all resources, including the KaTeX math renderer and its fonts, are served from this site itself, so no third party is contacted while you browse.
  • Learning progress (completed modules, quiz scores, theme choice) is stored only in your own browser using localStorage (keys ee-kb-progress-v1, ee-kb-theme). It is not transmitted anywhere. Clearing your browser data removes it.
  • Optional account for cross-device sync. If — and only if — you create an account, the following is stored on a server: your chosen username, a salted hash of your password, a salted hash of your recovery code, your learning progress, and timestamps. No email address and no other personal data are required or stored. The sync service runs on Cloudflare Workers with the database located in Western Europe; Cloudflare, Inc. transiently processes connection metadata (such as IP addresses) to deliver requests — see the Cloudflare privacy policy. The session token is kept in your browser's localStorage. You can delete your account yourself at any time in the account dialog; this immediately and permanently removes all your data from the server.
  • Hosting. The site is served via GitHub Pages. GitHub may process visitors' IP addresses in ordinary server logs — see the GitHub privacy statement.
